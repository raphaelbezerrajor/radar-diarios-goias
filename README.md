# Radar GO

Base editorial em HTML, CSS e JavaScript para monitoramento de diarios oficiais de Goias.

## O que tem aqui

- `radar-diarios-goias.html`: indice mensal.
- `radar-diarios-goias-cronologia.html`: visao cronologica.
- `radar-diarios-goias-dia.html`: pagina diaria parametrica.
- `radar-diarios-goias-data.js`: base principal consumida pela interface.
- `radar-diarios-goias-data.json`: espelho estruturado da base.
- `radar-diarios-goias.css`: estilos da interface.
- `radar-diarios-goias-app.js`: montagem dinamica das paginas.

## Como abrir

Basta abrir `radar-diarios-goias.html` no navegador.

## Escopo atual

- recorte mensal de abril de 2026;
- calendario por dia;
- cronologia;
- pagina diaria;
- primeiras entradas de Goiânia, Aparecida de Goiânia, Senador Canedo, Anápolis, AGM, MPGO, Estado de Goiás e TJGO.

## Observacao

O projeto esta em evolucao editorial. A curadoria ainda pode ser ampliada com mais leitura por palavra-chave, municipio, orgao, tipo de ato e secoes do DJE.
